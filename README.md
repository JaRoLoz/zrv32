# zrv32
32 bit Risc-V (rv32) emulator written over a Saturdy of boredom. Implemented instructions:
- add / addi
- sub
- sll / slli
- slt / slti
- sltu / sltiu
- xor / xori
- srl / srli
- sra / srai
- or / ori
- and / andi
- auipc
- beq / bne / blt / bge / bltu / bgeu
- jal / jalr
- lui
- lb / lh / lw / lbu / lhu
- sb / sh / sw
This is more of a PoC than a real emulator.
